## Updating the zigzag page

Edits are in **fab-public** on your Mac. To publish changes:

1. Open Terminal, go to the repo:  
   `cd ~/Documents/GitHub/fab-public`
2. Commit and push:  
   `git add shape-onboarding-journey && git commit -m "Update zigzag" && git push`

Or run from inside **shape-onboarding-journey**:  
`./push-updates.sh`  
(that script does the same from the repo root).

Site updates in 1–2 minutes after push.
