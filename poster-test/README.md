# Shape onboarding poster (test)

Zigzag “winding river” poster for Shape onboarding screens. Used to test layout and take screenshots.

**Live (after push):** https://janlindemans.github.io/fab-public/poster-test/

**Local test:** Open `index.html` in a browser, or run a screenshot:

```bash
cd poster-test
npx -y playwright install chromium
npx playwright screenshot index.html poster-capture.png --viewport-size=1200,900
```
